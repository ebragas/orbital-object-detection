Planetary Cargo Detector

This Python Flask application running on App Engine uses the Planetary Cargo Detector Deep Neural Network model on Google Cloud, as well as tools like Cloud DataStore and Cloud Storage to display the current system status as well as enable search on annotated satellite images.
